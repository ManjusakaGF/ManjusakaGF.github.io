Vue使用Scss

Sep 26 2018
Vue中使用Sass，需安装node-sass以及sass-loader，具体操作命令：
这里必须注意要先安装sass-loader，再安装node-sass，否则会很报多条错误信息
npm install --save-dev sass-loader
npm install --save-dev node-sass
这时不出意外就可以使用Scss了(Sass要严格按照语法来写)：
<style scoped lang="scss">
$color: #f00;
#id{
    color: $color;
}
</style>
